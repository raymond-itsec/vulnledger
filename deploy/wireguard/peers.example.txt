# WireGuard mesh peers - one per line.
#
# Format: NAME  WG_OVERLAY_IP  PUBLIC_ENDPOINT
#
#   NAME             matches keys/<NAME>.priv and keys/<NAME>.pub
#                    produced by generate-keys.sh
#   WG_OVERLAY_IP    the peer's IP on the WG overlay subnet (e.g. 10.99.0.X)
#   PUBLIC_ENDPOINT  HOST:PORT this peer accepts WG handshakes on (UDP).
#                    Default port is 51820. Use a single "-" for
#                    road-warrior peers (laptops, dynamic IPs) that
#                    initiate outbound only.
#
# Pick a /24 for the WG overlay that does not conflict with your other
# networks. RFC1918 ranges work; 10.99.0.0/24 is a common choice.
#
# To use:
#   1. cp peers.example.txt peers.txt
#   2. edit peers.txt to match your environment (gitignored)
#   3. make wg-bootstrap   (or: make wg-keys + make wg-render)
